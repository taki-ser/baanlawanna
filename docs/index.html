Baan Lawannaとは

"Baan"は「家」を意味するタイ語で、"Lawanna"は店主の名前です。Baan Lawannaという店名には、お客様をタイの別荘へご招待し、伝統的なマッサージを提供したいという意味が込められています。

Baan Lawannaは代々木公園で毎年開催されるタイフェスティバルにもNPO法人在日タイマッサージ協会の加盟店として参加しています。

店主 Lawanna

店主のLawanna(ラワンナ)はタイ北部のチェンライ県出身です。

タイ伝統医学協会が主催するタイマッサージおよびリフレクソロジーのトレーニングプログラムを修了しています。
